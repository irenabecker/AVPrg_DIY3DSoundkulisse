SoundSpace
Impressum

Audio-Video-Programming - Prof. Dr. Plaß & Sudau - WiSe 17/18

Project Descrpition:

„SoundSpace“ is a tangible sound application which allows you to transfer into different acoustic settings. As a user you are able to adjust every audio source the way you would like it to be.
You can use different shapes as well as a webinterface to adjust your SoundSpace. The shapes can be placed on a given plane, which represents the real room as a white box.
Using the webinterface the you can chooses a setting and set different parameters for each shape. Wherever you place the shapes, a Sound according to your Settings
in the web Interface will appear. Chooses a Setting as well as different Parameters for each shape.
By default every shape has already assigned an audiosource Just feel free to Experiment and enjoy your personal 3D SoundSpace!
